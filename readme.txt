The official wordoftravel plugin for travel bloggers publishing blog posts on wordoftravel.com

== Description ==

wordoftravel.com connects travel bloggers with real, independent travellers. This plugin streamlines the process of
categorising the topics and places associated with posts. Specifically, it allows registered travel bloggers to:

* Confirm ownership of their blog
* Add a link or tag to find their blog on wordoftravel
* Categorise the travel style and themes of their posts
* Select up to 3 places that the post is about

== Installation ==

To install the wordoftravel plugin:

1. Install the plugin through the WordPress plugins screen directly.
1. Activate the plugin through the 'Plugins' screen in WordPress
1. Use the Settings->wordoftravel screen to configure blog ownership or add links to wordoftravel
1. Use the wordoftravel post categorisation tab when writing or editing posts to add topics and places. 


== Frequently Asked Questions ==

= Can I use this plugin if I'm not signed up on wordoftravel? =

You can, but without an active listing on wordoftravel your posts won't be automatically indexed

= Does this plugin connect to any external sources? =

Yes, we use the publically available GeoNames Places API to enable you to add places and we connect to our own API when you click to verify blog ownership

= What is wordoftravel? =

wordoftravel helps independent travellers plan their trip. One of the ways we do this is by connecting travel bloggers with our audience. Travellers can search for blogs that cover the places they want to visit and the things they want to see and do. Learn more at [wordoftravel](https://wordoftravel.com)
